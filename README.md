# Combat Logs
Generate combat logs for machine learning project.


## CombatUnit
- Name
- Class
- Level
- Health
- Damage
- Offence
- Defense


## Simple D20 system
### Attack Roll
d20 + offence bonus

### Defend Roll
d20 + defense bonus

### Hit
attack > defend

### Miss
attack <= defend

### Damage Roll
d(dam_dice) for each level of the attacker
